概要

MAMPlayer 風の プレーヤー同梱のレンダラーです。ピアノロールのレンダリングを行います。
MAMPlayer ではピアノロールレンダリングのためのノート情報を事前演算しているようですが、このプラグインではリアルタイムに処理を行います。
そのため、MAMPlayer ほど精度が良くないです…。


ピアノロールの色を変えたい場合

以下の場所に設定ファイルを置いていますので、テキストエディタで編集することで変更可能です。

<MIDIPlayerの場所>/data/plugin/renderer/Pianoroll/config.xml

書式

<color midich = "?"  color = "#??????" />

midich: 対象となる MIDI ch (0〜15)
color: 色。 html 同様に #RRGGBB 形式が使用できます。
